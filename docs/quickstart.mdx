title: Quickstart
description: Run WordChord locally in a few minutes
---

## Prerequisites

- Node.js 18+
- Docker (for local Postgres)

## 1. Clone the repo

```bash
git clone https://github.com/nevobiton/Song-Notebook.git
cd Song-Notebook
```

## 2. Start the database

```bash
docker run -d --name song-pg \
  -e POSTGRES_DB=song_notebook \
  -e POSTGRES_USER=songuser \
  -e POSTGRES_PASSWORD=songpass \
  -p 54327:5432 postgres:16
```

## 3. Configure environment variables

```bash
cp server/.env.example server/.env
```

Open `server/.env` and fill in:

| Variable | Description |
|----------|-------------|
| `DATABASE_URL` | `postgresql://songuser:songpass@localhost:54327/song_notebook` |
| `JWT_SECRET` | Any long random string — run `openssl rand -hex 32` |
| `GOOGLE_CLIENT_ID` | From Google Cloud Console |
| `RESEND_API_KEY` | From resend.com |
| `APP_URL` | `http://localhost:5173` |
| `LOKI_URL` | Optional — see [Observability](/architecture#observability) |

## 4. Install and run

```bash
npm run install:all   # installs client + server dependencies
npm run dev           # starts frontend :5173 and backend :3001
```

## 5. Run migrations

```bash
cd server && npm run db:migrate
```

The app is now running at [http://localhost:5173](http://localhost:5173).

## Run tests

```bash
cd server && npm test
```
